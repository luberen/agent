# Development Container

## Build

```bash
./container/build.sh
```

## Run

```bash
./container/run.sh
```

## Interactive shell

```bash
./container/exec.sh
```

## Execute command

```bash
./container/exec.sh python3 --version
```

## Stop

```bash
./container/stop.sh
```

## Remove

```bash
./container/remove.sh
```
